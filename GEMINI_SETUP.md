# Настройка Gemini AI для автогенерации информации о поэтах

## Шаги настройки:

### 1. Получите API ключ Gemini

1. Перейдите на [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Войдите с вашим Google аккаунтом
3. Нажмите "Create API Key"
4. Скопируйте полученный API ключ

### 2. Создайте файл `.env`

В корневой папке проекта создайте файл `.env` и добавьте ваш API ключ:

```env
VITE_GEMINI_API_KEY=ваш_api_ключ_здесь
```

**Важно:** Файл `.env` уже добавлен в `.gitignore`, поэтому ваш API ключ не попадет в репозиторий.

### 3. Перезапустите приложение

После создания файла `.env` перезапустите dev сервер:

```bash
npm run dev
```

## Как использовать

1. Откройте форму добавления нового поэта
2. Введите имя поэта (например, "Лермонтов")
3. Нажмите кнопку **"AI ✨"** рядом с полем "Досье"
4. AI автоматически сгенерирует информацию о поэте и заполнит поле
5. Вы можете отредактировать сгенерированный текст перед сохранением

## Примечания

- Генерация занимает несколько секунд
- Во время генерации поле "Досье" будет заблокировано
- Если API ключ не настроен, вы увидите сообщение об ошибке
- Можно продолжать добавлять поэтов вручную без AI, просто вводя информацию напрямую

## Устранение проблем

Если генерация не работает:

1. Проверьте, что файл `.env` создан в корне проекта
2. Проверьте, что API ключ скопирован правильно (без пробелов)
3. Проверьте интернет-соединение
4. Убедитесь, что вы перезапустили dev сервер после создания `.env`
5. Проверьте консоль браузера на наличие ошибок (F12 → Console)
